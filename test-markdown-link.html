🔧 技术实现

修改后的实现使用了跨平台的系统命令：

  • macOS: open "url" - 在默认浏览器中打开链接
  • Windows: start "" "url" - 在默认浏览器中打开链接
  • Linux: xdg-open "url" - 在默认浏览器中打开链接

这些命令会优先在已有的浏览器实例中打开新标签页，而不是启动新的浏览器窗口。

  2. 在应用中打开包含链接的 Markdown 内容
  3. 点击链接
  4. 观察是否在已有浏览器窗口中打开了新标签页
